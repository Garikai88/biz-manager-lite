# Biz-Manager Lite

### Smart Task Manager & Productivity Ledger

Biz-Manager Lite is a web-based productivity and project management application built for freelancers, software developers, and small remote teams. It unifies project tracking, task sprint management, and basic financial analytics into a single dashboard.

## 🛠️ Technology Stack

- **Frontend:** HTML5, CSS3 (Flexbox & Grid)
- **Scripting:** Modular JavaScript (ES6 Modules)
- **Data Storage:** Client-side LocalStorage
- **Third-Party APIs:** Trello REST API, CoinGecko API

## 📂 Project Architecture

This application utilizes a clean separation of concerns using functional modules. The `js/` directory isolates core state management from DOM rendering tools, while dedicated wrappers in `js/services/` manage network fetch commands cleanly.
